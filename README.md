# E-Commerce SQL Analytics Project

This project focuses on analyzing a real-world Brazilian E-Commerce dataset using SQL and MySQL Workbench. The objective of this project was to perform business-focused data analysis using relational databases and advanced SQL concepts.

The project includes:

* Multi-table joins
* Customer and revenue analysis
* Product and payment analysis
* Business KPI analysis
* Window functions
* CTEs (Common Table Expressions)
* CASE WHEN logic
* Advanced aggregations
* ER Diagram modeling

## Tools Used

* SQL
* MySQL Workbench
* GitHub

## SQL Concepts Used

* INNER JOIN & LEFT JOIN
* GROUP BY & HAVING
* Aggregate Functions
* CASE WHEN
* Window Functions
* CTEs
* KPI Analysis

## Business Analysis Performed

* Customer Segmentation
* Revenue Analysis
* Payment Behavior Analysis
* Monthly Sales Trends
* Repeat Customer Analysis
* Product Category Performance
* Customer Lifetime Value Estimation

## Dataset

Brazilian Olist E-Commerce Dataset from Kaggle.

## Project Goal

The goal of this project was to strengthen practical SQL skills while generating meaningful business insights from real-world relational datasets.
